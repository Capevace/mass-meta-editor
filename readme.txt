== Description ==

With this plugin you can easily edit SEO metadata, that is provided by the Yoast SEO plugin, in bulk.
It is meant as an improvement/replacement to the bulk editor built into the Yoast SEO plugin.
Please note, that this plugin is in no way associated with the Yoast SEO plugin or Yoast BV.

== Installation ==

1. Download the plugin and upload the .zip-File it to your WordPress site.
2. The Editor is now available in the Sidebar under Tools/Mass Meta Editor.

Updates show up on the WordPress Plugins page automatically, so you'll never need to reinstall the plugin.

== Frequently Asked Questions ==

= Is this plugin associated with Yoast SEO? =

No. While this plugin requires the Yoast SEO plugin to be used, it is not made in correspondence with Yoast BV.
